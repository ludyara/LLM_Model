Этот проект посвящён анализу тональности текстовых отзывов с использованием методов машинного обучения и предобученных языковых моделей (LLM).

В рамках проекта реализован pipeline:

- сбор данных (парсинг отзывов с веб-сайта),
- формирование датасета,
- ручная разметка,
- обучение модели классификации,
- оценка качества на тестовой выборке.

Как это работает
1. Сбор данных
Отзывы парсятся с веб-страниц с использованием requests / Selenium и BeautifulSoup
Из HTML извлекаются тексты отзывов:
<a class="review-title">Текст отзыва</a>

Все отзывы сохраняются в CSV:

2. Разметка данных
Все отзывы изначально помечаются как positive
Негативные отзывы размечаются вручную

3. Преобразование текста (LLM embeddings)
Используется модель sentence-transformers (all-MiniLM-L6-v2)
Каждый текст преобразуется в вектор (embedding), отражающий его смысл.

4. Обучение модели
Используется классический алгоритм Logistic Regression. Обучение проводится на embeddings, а не на “сырых” текстах.
Для борьбы с дисбалансом классов применяется class_weight="balanced"

6. Разделение выборки
Данные делятся на обучающую (80%) и тестовую (20%) с сохранением пропорций классов

6. Оценка качества
Используются метрики:
- precision
- recall
- f1-score
- accuracy

Результат получился следующий:

1       0.80      0.84      0.82        19

accuracy = 0.82

Используемые технологии: Python, pandas, scikit-learn, sentence-transformers, BeautifulSoup, Selenium
Особенности и сложности: сайт использует защиту от ботов (капча), для решения применялся Selenium
